NetworkPlot

A d3-based network plot.

Example

<NetworkPlot
    width={400}
    height={400}
    data={{
        nodes: [
            { 'id': 0 },
            { 'id': 1 },
            { 'id': 2 },
            { 'id': 3 },
        ],
        links: [
            { 'source': 0, 'target': 1 },
            { 'source': 0, 'target': 2 },
            { 'source': 2, 'target': 3 }
        ]
    }}
/>

Open interactive preview

Options

  • width | number: width of the plot (in px). Default: 900.
  • height | number: height of the plot (in px). Default: 600.
