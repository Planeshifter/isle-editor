A normal distribution QQ plot.

## Options

* __data__ | `object (required)`: object of value arrays. Default: `none`.
* __variable__ | `string (required)`: variable to display. Default: `none`.


## Examples

```jsx live
<QQPlot 
    data={heartdisease} 
    variable="Age"
/>
```
