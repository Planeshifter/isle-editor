A learning component illustrating conditional probability for two dependent binary variables.

## Options

* __A__ | `string`: first variable. Default: `'A'`.
* __B__ | `string`: second variable. Default: `'B'`.


## Examples

```jsx live
<LearnConditionalProbability />
```
