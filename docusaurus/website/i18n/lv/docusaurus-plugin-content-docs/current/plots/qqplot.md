Normālā sadalījuma QQ diagramma.

## Iespējas

* __data__ | `object (required)`: vērtību masīvu objekts. Default: `none`.
* __variable__ | `string (required)`: mainīgais lielums, lai parādītu. Default: `none`.


## Piemēri

```jsx live
<QQPlot 
    data={heartdisease} 
    variable="Age"
/>
```
