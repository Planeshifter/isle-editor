Elements, kas attēlo iepriekš formatētu tekstu.

## Iespējas

* __className__ | `string`: klases nosaukums. Default: `''`.
* __style__ | `object`: CSS iebūvētie stili. Default: `{}`.


## Piemēri

```jsx live
<pre>
    mean(x) - mean function
</pre>
```
