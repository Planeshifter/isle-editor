## Iespējas

* __hideCDF__ | `boolean`: kontrolē, vai slēpt CDF displeju.. Default: `false`.
* __step__ | `(number|string)`: ritināšanas ieejas soļa lielums. Default: `0.01`.
* __style__ | `object`: CSS iebūvētie stili. Default: `{}`.
