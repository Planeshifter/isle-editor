Élément générique du conteneur.

## Options

* __className__ | `string`: nom de la classe. Default: `''`.
* __style__ | `object`: Styles CSS en ligne. Default: `{}`.


## Examples

```jsx live
<div>
    Use a div as a generic container for other content.
</div>
```
