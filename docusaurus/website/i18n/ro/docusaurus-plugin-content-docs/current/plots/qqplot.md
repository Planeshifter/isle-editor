Un grafic QQ cu distribuție normală.

## Opțiuni

* __data__ | `object (required)`: obiect de matrice de valori. Default: `none`.
* __variable__ | `string (required)`: variabilă pentru a afișa. Default: `none`.


## Exemple

```jsx live
<QQPlot 
    data={heartdisease} 
    variable="Age"
/>
```
