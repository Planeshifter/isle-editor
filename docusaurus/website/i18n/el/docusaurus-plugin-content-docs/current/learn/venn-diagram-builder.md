Ένας κατασκευαστής διαγραμμάτων Venn.

## Επιλογές

* __nobs__ | `number`: συνολικός αριθμός παρατηρήσεων. Default: `none`.
* __three__ | `boolean`: ελέγχει αν θα εμφανίζονται τρεις κύκλοι. Default: `false`.


## Παραδείγματα

```jsx live
<LearnVennDiagramBuilder />
```
