Μια συνιστώσα μάθησης που απεικονίζει την υπό συνθήκη πιθανότητα για δύο εξαρτημένες δυαδικές μεταβλητές.

## Επιλογές

* __A__ | `string`: πρώτη μεταβλητή. Default: `'A'`.
* __B__ | `string`: δεύτερη μεταβλητή. Default: `'B'`.

