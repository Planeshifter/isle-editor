Στοιχείο που εμφανίζει προσχηματισμένο κείμενο.

## Επιλογές

* __className__ | `string`: όνομα κλάσης. Default: `''`.
* __style__ | `object`: Στυλ γραμμής CSS. Default: `{}`.


## Παραδείγματα

```jsx live
<pre>
    mean(x) - mean function
</pre>
```
