Διάγραμμα QQ κανονικής κατανομής.

## Επιλογές

* __data__ | `object (required)`: αντικείμενο των πινάκων τιμών. Default: `none`.
* __variable__ | `string (required)`: μεταβλητή για την εμφάνιση. Default: `none`.


## Παραδείγματα

```jsx live
<QQPlot 
    data={heartdisease} 
    variable="Age"
/>
```
