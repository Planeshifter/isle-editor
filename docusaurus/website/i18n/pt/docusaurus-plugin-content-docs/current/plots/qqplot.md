Função de comparação utilizada para ordenar valores em ordem ascendente.

## Opções

* __data__ | `object (required)`: objeto de arrays de valor. Default: `none`.
* __variable__ | `string (required)`: variável a exibir. Default: `none`.


## Exemplos

```jsx live
<QQPlot 
    data={heartdisease} 
    variable="Age"
/>
```
