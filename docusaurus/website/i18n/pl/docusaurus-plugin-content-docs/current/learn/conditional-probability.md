Komponent edukacyjny ilustrujący warunkowe prawdopodobieństwo dla dwóch zależnych od siebie zmiennych binarnych.

## Opcje

* __A__ | `string`: pierwsza zmienna. Default: `'A'`.
* __B__ | `string`: druga zmienna. Default: `'B'`.
