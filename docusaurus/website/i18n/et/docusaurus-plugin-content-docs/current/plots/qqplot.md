Normaalse jaotuse QQ graafik.

## Valikud

* __data__ | `object (required)`: väärtuste massiivi objekt. Default: `none`.
* __variable__ | `string (required)`: muutuja kuvamiseks. Default: `none`.


## Näited

```jsx live
<QQPlot 
    data={heartdisease} 
    variable="Age"
/>
```
