Ein Venn-Diagramm-Ersteller.

## Options

* __nobs__ | `number`: Gesamtzahl der Beobachtungen. Default: `none`.
* __three__ | `boolean`: steuert, ob drei Kreise angezeigt werden sollen. Default: `false`.
