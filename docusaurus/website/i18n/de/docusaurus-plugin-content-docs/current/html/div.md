Generisches Container-Element.

## Options

* __className__ | `string`: Klassenname. Default: `''`.
* __style__ | `object`: CSS-Inline-Stile. Default: `{}`.


## Examples

```jsx live
<div>
    Use a div as a generic container for other content.
</div>
```
