Graf QQ s normálnym rozdelením.

## Možnosti

* __data__ | `object (required)`: objekt polí hodnôt. Default: `none`.
* __variable__ | `string (required)`: premenná na zobrazenie. Default: `none`.


## Príklady

```jsx live
<QQPlot 
    data={heartdisease} 
    variable="Age"
/>
```
