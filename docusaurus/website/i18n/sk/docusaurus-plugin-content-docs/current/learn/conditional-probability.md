Komponent učenia znázorňujúci podmienenú pravdepodobnosť pre dve závislé binárne premenné.

## Možnosti

* __A__ | `string`: prvá premenná. Default: `'A'`.
* __B__ | `string`: druhá premenná. Default: `'B'`.
