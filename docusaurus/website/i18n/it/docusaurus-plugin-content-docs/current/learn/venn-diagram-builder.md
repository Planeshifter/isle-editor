Un costruttore di diagrammi di Venn.

## Opzioni

* __nobs__ | `number`: numero totale di osservazioni. Default: `none`.
* __three__ | `boolean`: controlla se visualizzare tre cerchi. Default: `false`.


## Esempi

```jsx live
<LearnVennDiagramBuilder />
```
