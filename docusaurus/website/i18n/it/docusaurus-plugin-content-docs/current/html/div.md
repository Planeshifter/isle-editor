Elemento generico del contenitore.

## Options

* __className__ | `string`: nome della classe. Default: `''`.
* __style__ | `object`: Stili in linea CSS. Default: `{}`.


## Examples

```jsx live
<div>
    Use a div as a generic container for other content.
</div>
```
