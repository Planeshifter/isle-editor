Elemento che visualizza il testo preformattato.

## Opzioni

* __className__ | `string`: nome della classe. Default: `''`.
* __style__ | `object`: Stili in linea CSS. Default: `{}`.


## Esempi

```jsx live
<pre>
    mean(x) - mean function
</pre>
```
