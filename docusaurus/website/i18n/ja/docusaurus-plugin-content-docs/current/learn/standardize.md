データの標準化のプロセスを説明する学習コンポーネント。

## Options

* __showProbabilities__ | `boolean`: 確率を表示するかどうかを制御します。. Default: `false`.
* __step__ | `object`: スクロール入力のステップサイズ. Default: `'any'`.


## Examples

```jsx live
<LearnStandardize />
```
