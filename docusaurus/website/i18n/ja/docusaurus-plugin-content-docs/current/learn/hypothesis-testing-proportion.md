母集団の割合に対する仮説検定に関する学習コンポーネント。

## Options

* __types__ | `array<string>`: ウィジェットが公開するテストのタイプ (`1サンプル`, `2サンプル`) を指定します。. Default: `[
  'One-Sample',
  'Two-Sample'
]`.
* __feedback__ | `boolean`: フィードバックボタンを表示するかどうかを制御します。. Default: `false`.
* __style__ | `object`: CSSインラインスタイル. Default: `{}`.


## Examples

```jsx live
<LearnProportionTest />
```
