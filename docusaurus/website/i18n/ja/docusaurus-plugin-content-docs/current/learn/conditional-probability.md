2つの従属バイナリ変数の条件付き確率を説明する学習コンポーネント。

## オプション

* __A__ | `string`: 第一変数. Default: `'A'`.
* __B__ | `string`: だいにへんすう. Default: `'B'`.


## 例としては、以下のようなものがあります。

```jsx live
<LearnConditionalProbability />
```
