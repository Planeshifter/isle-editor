現在の時刻を表示する時計コンポーネント。

## Options

* __duration__ | `boolean`: 設定されている場合、セッションの持続時間が表示されます (実際の時間ではありません). Default: `false`.
* __format__ | `string`: じかんけいしき. Default: `''`.
* __className__ | `string`: クラス名. Default: `''`.
* __style__ | `object`: CSSインラインスタイル. Default: `{}`.


## Examples

```jsx live
<Clock />
```
