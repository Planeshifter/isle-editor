Komponenta učení ilustrující podmíněnou pravděpodobnost pro dvě závislé binární proměnné.

## Možnosti

* __A__ | `string`: první proměnná. Default: `'A'`.
* __B__ | `string`: druhá proměnná. Default: `'B'`.
