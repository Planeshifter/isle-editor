Oppimiskomponentti, joka havainnollistaa kahden riippuvaisen binäärimuuttujan ehdollista todennäköisyyttä.

## Vaihtoehdot

* __A__ | `string`: ensimmäinen muuttuja. Default: `'A'`.
* __B__ | `string`: toinen muuttuja. Default: `'B'`.
