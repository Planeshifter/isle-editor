Venn-diagrammin rakentaja.

## Vaihtoehdot

* __nobs__ | `number`: havaintojen kokonaismäärä. Default: `none`.
* __three__ | `boolean`: säädetään, näytetäänkö kolme ympyrää. Default: `false`.
