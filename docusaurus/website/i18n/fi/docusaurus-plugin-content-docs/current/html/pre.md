Elementti, joka näyttää esimuotoiltua tekstiä.

## Vaihtoehdot

* __className__ | `string`: luokan nimi. Default: `''`.
* __style__ | `object`: CSS-inline-tyylit. Default: `{}`.


## Esimerkkejä

```jsx live
<pre>
    mean(x) - mean function
</pre>
```
