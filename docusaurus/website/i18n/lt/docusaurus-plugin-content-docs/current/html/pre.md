Elementas, kuriame rodomas iš anksto suformatuotas tekstas.

## Parinktys

* __className__ | `string`: klasės pavadinimas. Default: `''`.
* __style__ | `object`: CSS eilutės stiliai. Default: `{}`.


## Pavyzdžiai

```jsx live

    mean(x) - mean function
</pre>
```
