En normalfördelad QQ-plott.

## Alternativ

* __data__ | `object (required)`: objekt av värdefält. Default: `none`.
* __variable__ | `string (required)`: variabel för att visa. Default: `none`.


## Exempel

```jsx live
<QQPlot 
    data={heartdisease} 
    variable="Age"
/>
```
