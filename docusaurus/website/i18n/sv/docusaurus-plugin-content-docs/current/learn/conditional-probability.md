En inlärningskomponent som illustrerar villkorlig sannolikhet för två beroende binära variabler.

## Alternativ

* __A__ | `string`: första variabeln. Default: `'A'`.
* __B__ | `string`: andra variabeln. Default: `'B'`.


## Exempel

```jsx live
<LearnConditionalProbability />
```
