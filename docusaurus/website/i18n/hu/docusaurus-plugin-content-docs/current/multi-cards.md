Egy ISLE komponens, amely lehetővé teszi, hogy több felfordítható kártyát hozzon létre és irányítson (pl. egy Memory játékban).

## Opciók

* __animation__ | `object`: ha be van állítva, a komponens egy belépő animációt használ; az objektum tartalmaz egy nevet (pl. `anim-scale-up`) és egy időtartamot (pl. `1.7s` = 1.7 másodperc).. Default: `{}`.
* __memory__ | `boolean`: engedélyezi-e a memóriajáték módot, amelyben a megadott értékek párjait kell összepárosítani.. Default: `none`.
* __onChange__ | `function`: egy függvény, amely megkapja a lapozható kártyák mátrixát. Default: `onChange() {}`.
* __onMemoryComplete__ | `function`: visszahívás, ha a memóriajáték befejeződött. Default: `onMemoryComplete() {}`.
* __oneTime__ | `boolean`: jelzi, hogy a flip folyamat csak egyszer hajtható-e végre.. Default: `false`.
* __values__ | `array<object>`: a megfelelő kártyák értékei, egy olyan tömbben, amely egy "elülső" és egy "hátsó" értéket tartalmazó mezővel rendelkezik. Ilyen érték lehet egy karakterlánc, de lehet egy teljes értékű ISLE komponens is.. Default: `[]`.
* __style__ | `object`: CSS inline stílusok. Default: `{}`.
* __containerStyle__ | `object`: A kártya konténerek CSS stílusa. Default: `{}`.
* __frontStyle__ | `object`: A kártyák előlapjának CSS stílusa. Default: `{}`.
* __backStyle__ | `object`: A kártyák hátoldalának CSS stílusa. Default: `{}`.
