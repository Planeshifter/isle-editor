Компонент обучения, объясняющий процесс стандартизации данных.

## Варианты

* __showProbabilities__ | `boolean`: управляет отображением вероятностей. Default: `false`.
* __step__ | `(number|string)`: размер шага прокрутки. Default: `'any'`.
