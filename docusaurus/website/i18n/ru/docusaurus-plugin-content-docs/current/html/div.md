Элемент общего контейнера.

## Options

* __className__ | `string`: название класса. Default: `''`.
* __style__ | `object`: встроенные стили CSS. Default: `{}`.


## Examples

```jsx live
<div>
    Use a div as a generic container for other content.
</div>
```
