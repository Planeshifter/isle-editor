Элемент, отображающий предварительно отформатированный текст.

## Options

* __className__ | `string`: название класса. Default: `''`.
* __style__ | `object`: встроенные стили CSS. Default: `{}`.


## Examples

```jsx live
<pre>
    mean(x) - mean function
</pre>
```
