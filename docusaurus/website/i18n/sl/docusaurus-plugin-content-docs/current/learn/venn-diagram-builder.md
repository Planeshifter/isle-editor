Gradnik Vennovega diagrama.

## Možnosti

* __nobs__ | `number`: skupno število opazovanj. Default: `none`.
* __three__ | `boolean`: nadzoruje, ali se prikažejo trije krogi.. Default: `false`.


## Primeri

```jsx live
<LearnVennDiagramBuilder />
```
